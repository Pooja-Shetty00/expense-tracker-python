# Expense Tracker

A Python-based expense management application that allows users to track, manage, and analyze their daily expenses.

## Features

* Add expenses
* View all expenses
* Calculate total expenses
* Delete expenses
* Category-wise expense summary
* Save expenses to file
* Load expenses automatically

## Technologies Used

* Python
* File Handling
* Lists
* Dictionaries

## How to Run

python3 expense_tracker.py

## Future Improvements

* Edit expenses
* Monthly reports
* Graphical User Interface (GUI)
* CSV Export
